교회 주보 변환 시스템

🌍 다국어 지원

  • 한국어 - 원본 텍스트
  • 영어 - English
  • 중국어 - 简体中文
  • 일본어 - 日本語
  • 인도네시아어, 스페인어, 러시아어, 프랑스어

📖 인터랙티브 기능

  • 성경 구절 클릭 → 다국어 본문 팝업
  • 찬송가 번호 클릭 → 다국어 가사 팝업
  • 설교 본문 전체 번역
  • 오늘의 양식 다국어 지원

📱 주보 섹션 구조

  • 오늘의 말씀 (컴팩트 아코디언)
  • 예배 안내 (공통순서 + 개별 카드)
  • 생명의 말씀 (설교 아코디언)
  • 금주의 찬양 / 교회소식 / 헌금 안내

🛠️ 기술 스택

  • Python Flask + Claude Vision API
  • PyMuPDF (PDF 처리)
  • 다국어 폴백: 선택어 → 영어 → 한국어
  • 반응형 CSS + 다크모드

  • "Wi-Fi 비밀번호를 안내해 드릴게요"

  • 온라인 + 오프라인 통합 유통환경
  • 도매점 → 소매점 → 소비자 3단계 연계
  • 카카오톡 기반 모바일 주문 시스템
  • 실시간 재고/정산 통합 관리
