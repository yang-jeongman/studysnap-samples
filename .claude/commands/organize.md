프로젝트 파일을 정리합니다.

## 정리 대상
$ARGUMENTS

## 기본 작업
1. outputs/ 폴더의 임시 파일 확인
2. 중복 파일 식별
3. 네이밍 규칙에 맞지 않는 파일 찾기
4. 빈 폴더 확인

## 표준 구조
```
outputs/
├── Church/{교회명}/{날짜}.html
├── Lecture/{과목}/{주제}.html
└── Election/{후보명}/{날짜}.html
```

## 네이밍 규칙
- 교회 주보: {교회명}_{날짜}_모바일주보.html
- 강의자료: {과목}_{주제}_{날짜}.html
- 선거공보: {후보명}_{날짜}.html

파일 목록을 확인하고 정리 방안을 제안해주세요.
