최근 변경된 코드를 리뷰해주세요.

## 리뷰 범위
$ARGUMENTS

## 리뷰 항목
1. **버그 및 로직 오류** - 잠재적 버그, 엣지 케이스
2. **보안 취약점** - SQL 인젝션, XSS, CSRF 등
3. **성능 이슈** - 비효율적인 코드, N+1 쿼리
4. **코드 스타일** - 네이밍, 구조, 가독성

## 출력 형식
- Critical: 즉시 수정 필요
- Major: 수정 권장
- Minor: 개선 가능

git diff로 변경사항을 먼저 확인하고 리뷰해주세요.
