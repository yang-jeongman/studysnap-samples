# Quality Check Command

사용자가 품질 검증을 요청했습니다.

## 실행 단계

1. **대상 파일 확인**
   - 인자로 전달된 HTML 파일 경로 확인
   - 없으면 가장 최근 변환된 파일 선택

2. **품질 검증 파이프라인 실행**
   - OCR 정확도 검증 (ocr-accuracy-verifier)
   - 콘텐츠 무결성 검증 (content-validator)
   - 스타일 일관성 검증
   - 번역 품질 검증

3. **결과 보고**
   ```
   ## 품질 검증 결과

   전체 점수: XX%

   | 항목 | 점수 | 상태 |
   |------|------|------|
   | OCR 정확도 | XX% | OK/경고/오류 |
   | 콘텐츠 무결성 | XX% | OK/경고/오류 |
   | 스타일 일관성 | XX% | OK/경고/오류 |
   | 번역 품질 | XX% | OK/경고/오류 |

   ### 발견된 문제
   - [오류/경고 목록]

   ### 자동 교정 적용
   - [교정 내역]
   ```

4. **학습 피드백 저장**
   - 오류 패턴 기록
   - 다음 변환 개선에 반영

## 사용 예시
```
/quality-check
/quality-check outputs/Church/여의도순복음교회/2025-12-19.html
/quality-check --all outputs/Church/
```
