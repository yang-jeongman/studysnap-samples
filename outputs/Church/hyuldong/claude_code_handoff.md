- 반응형 모바일 최적화 디자인

**기술 스택:**
- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Noto Serif KR, Noto Sans KR)
- CSS Variables 기반 테마 시스템
- 애니메이션 (fadeIn, scale transitions)

**디자인 컨셉:**
- 따뜻한 브라운/골드 컬러 팔레트
- 세리프 + 산세리프 폰트 조합
- 카드 기반 UI
