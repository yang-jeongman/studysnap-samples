# 송노섭 당진시장 후보 가상 선거공보물

## 📋 프로젝트 개요

본 프로젝트는 송노섭 교수의 가상 당진시장 선거 공보물을 AI 기반으로 생성한 샘플입니다.

### 🎯 제작 목적
- 기존 선거 공보물 샘플들을 분석하여 새로운 형태의 공보물 제작
- PDF 없이 후보자 정보를 바탕으로 직접 HTML 선거공보물 생성
- 지방선거(시장 선거)에 맞춤화된 공약 및 구성

## 👤 후보자 정보

### 기본 정보
- **이름**: 송노섭
- **직책**: 당진시장 후보 (가상)
- **정당**: 더불어민주당
- **학력**:
  - 단국대학교 법학 학사
  - 단국대학교 경영학 석사
  - 단국대학교 경영학 박사

### 주요 경력
- 송영진 국회의원 보좌관
- 일양약품(주) 충남사업본부장
- 단국대학교 초빙교수
- 충남도계 및 당진땅 수호 범시민대책위원회 조직위원장

### 특징
- 🏃 마라톤 풀코스 30회 이상 완주 (뚝심과 끈기)
- 💼 정치·경영·학계 경험을 고루 갖춘 준비된 리더
- 🏛️ 당진 지역 활동 10년 이상 (JC, 로타리, 번영회 등)

## 🎯 핵심 공약 (6대 분야)

### 1. 경제는 활기차게 - 산업과 일자리
- 석문산업단지 및 송산2산단 활성화
- 제약·바이오산업 유치
- 중소기업 지원 확대
- 당진항 운영 공사화

### 2. 농축수산업 안정화 - 1차산업 육성
- 쌀값 안정 대책 마련
- FTA 대비 농업정책
- 신선채소단지 조성
- 바다목장 육성

### 3. 복지는 따뜻하게 - 사회안전망
- 노인복지공동체 형성
- 시니어 일자리 창출
- 장애인 복지센터 건립
- 청년실업 해결

### 4. 교육은 희망차게 - 미래인재
- 산학융합형 특성화 학과
- 공공보육시설 확충
- 산업인력 전문교육기관
- 고교 무상교육

### 5. 환경은 쾌적하게 - 지속가능성
- 대기환경 개선 특별지구
- 삽교호·석문호 수질개선
- 철탑·송전선로 대응
- 신재생에너지 확대

### 6. 주민은 편안하게 - 인프라
- 천안-당진간 고속도로
- 석문공단 산업철도
- 종합병원 유치
- 문화예술공간 마련
